样式写法对比：Tailwind 原子类 vs MUI sx prop

⚠️ MUI sx prop（runtime CSS-in-JS）
// 需要理解 MUI 主题 token 才能读懂
<Button
  variant="contained"
  sx={{
    bgcolor: 'grey.900',
    borderRadius: 2,
    px: 2, py: 1,
    '&:hover': {
      bgcolor: 'grey.700'
    }
  }}
>
  下一关
</Button>

// AI 修改时须理解 MUI spacing(2)=16px
// grey.900 需查 MUI 主题才知道是 #212121
runtime emotion · 需主题上下文 · AI易出错
✅ Tailwind 原子类（编译期消除）
// 每个 class 即文档，无需上下文
<button
  className="
    bg-slate-900
    hover:bg-slate-700
    rounded-lg
    px-4 py-2
    text-white text-sm
    transition-colors
  "
>
  下一关
</button>

// AI 直接 grep className 即可理解样式
// 编译后只保留用到的类，零 runtime
编译期消除 · 自文档 · AI 精确可控
